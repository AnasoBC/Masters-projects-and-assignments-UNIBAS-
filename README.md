# Banking and Fintech Revolution-
Fintech Lenders in the U.S. Mortgage Market — University of Basel, 2025
Empirical study of fintech vs. traditional mortgage lenders using 28,000+ Freddie Mac loan-level records. Built Random Forest classifiers for delinquency (FICO: AUC 0.73; DTI: AUC 0.61) and lender-stratified interest-rate models (fintech AUC 0.71 vs. non-fintech 0.64). Co-authored with P. Nangole-Meyer.
Tools: Python, pandas, scikit-learn, seaborn, plotly
